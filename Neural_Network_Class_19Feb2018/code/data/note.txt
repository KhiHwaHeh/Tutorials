ကိုယ်က classification လုပ်ချင်တဲ့ ပုံတွေကို ./data ဖိုလ်ဒါအောက်မှာ class တစ်ခုချင်းစီအလိုက် ခွဲပြီး သိမ်းပါ။

UCSY Neural Network class မှာ ရိုက်ယူခဲ့တဲ့၊ စမ်းခဲ့တဲ့ Myanmar fingerspelling ပုံတွေကို တင်မထားပါဘူး။
သို့သော် feature ဖိုင်ကိုလည်း မြင်ရအောင်၊ id-name dictionary ဖိုင်ကိုလည်း လေ့လာလို့ ရအောင် တင်ပေးထားပါတယ်။
အဲဒါကြောင့် training, testing အဆင့်တွေက ချက်ခြင်း လုပ်လို့ ရပါလိမ့်မယ်။

feature extraction အဆင့်ကနေ စလုပ်ဖို့အတွက်ဆိုရင်တော့ ဓာတ်ပုံတွေကို ရိုက်ယူတာမျိုး၊ သို့မဟုတ် ရှိတဲ့ ပုံတွေကိုသုံးတာမျိုးကတော့ အလေ့အကျင်ရအောင် ကိုယ်တိုင် လုပ်ကြည့်စေချင်ပါတယ်။
