ကိုယ်က classification လုပ်ချင်တဲ့ ပုံတွေကို ./data ဖိုလ်ဒါအောက်မှာ class တစ်ခုချင်းစီအလိုက် ခွဲပြီး သိမ်းပါ။

အဆင်သင့်စမ်းrun ချင်ရင် လည်း အဆင်ပြေအောင်လို့ UCSY Neural Network class မှာ ရိုက်ယူခဲ့တဲ့၊ စမ်းခဲ့တဲ့ Myanmar fingerspelling ပုံတွေကို (က ကနေ င အထိ) တင်ပေးထားပါတယ်။ ဒီနေရာ့မှာ .f ဖိုင်တွေက ပုံတစ်ပုံချင်းအတွက် feature ဖိုင်ဖြစ်ပါတယ်။

feature ထုတ်တာက အချိန်နည်းနည်းစောင့်ရပါလိမ့်မယ်။

==========

ကောင်းတာကတော့ ဒေတာပြင်ဆင်တဲ့ အဆင့် (ဓာတ်ပုံရိုက်ယူတာမျိုး)၊ သို့မဟုတ် ရှိနေတဲ့ ပုံတွေကို ကိုယ်တိုင် အုပ်စုခွဲကြည့်ပြီး၊ feature extraction အဆင့်ကနေ အလေ့အကျင်ရအောင် ကိုယ်တိုင် လုပ်ကြည့်စေချင်ပါတယ်။ အဲဒါမှ တဆင့် run ပြီးသွားရင် ဘယ်ဖိုင်တွေက output အဖြစ်ရရှိလာတယ်ဆိုတာရယ်၊ preprocessing မှာ ကြုံရတဲ့ လက်တွေ့ ပြဿနာတွေကိုလည်း မြင်လာမှာ မို့ပါ။
